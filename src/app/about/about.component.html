SaleTrackr

Willkommen bei SaleTrackr! Hier können Sie Ihre Bestellungen verwalten.

Aber wie funktioniert das?

  • 1. Ihre Hauptseite
  • 2. Neue Verkäufe registrieren
  • 3. Einen Verkauf löschen
  • 4. Einen Eintrag ändern

1. Ihre Hauptseite

In Ihrer persönlichen Startseite "Home" sehen Sie Ihre aktuellsten Verkäufe.
Standardmäßig werden dort Ihre Verkäufe der letzten 7 Tage angezeigt. Sollte es in diesem Zeitraum keine Verkäufe gegeben haben, so wird Ihnen der letzte Verkauf - zeitunabhängig angezeigt.
Darunter sehen Sie - rechts unten - einen kleinen Zähler, der Ihnen zeigt, wie viele Verkäufe in den letzten 7 Tagen getätigt wurden.

2. Neue Verkäufe registrieren

Wenn Sie auf einer Verkaufsplattform Artikel verkauft haben, können Sie ganz einfach dieses Tool nutzen und über einen Klick auf "Add Order" in der obigen Navigationsleiste einen neuen Eintrag in Ihre Datenbank anlegen, welcher alle Infos über Ihren Verkauf speichern kann.

3. Details eines Verkaufs einsehen

Möchten Sie Details eines Verkaufs einsehen, so können Sie dies durch Klick auf das Register Orders erledigen.
Dort sehen Sie alle Einträge, die Sie bisher in Ihre Datenbank gespeichert haben.
Möchten Sie nun die Details über diese Verkäufe einsehen, so klicken Sie jeweils in der Karte des jeweiligen Verkaufs auf das Icon i, woraufhin sich weitere Details des Verkaufs öffnen.

4. Einen Verkauf löschen

Möchten Sie einen Verkauf doch aus der Datenbank entfernen, so können Sie das im Register Orders erledigen.
Dazu suchen Sie den Eintrag, den Sie löschen möchten und klicken in diesem Eintrag auf das Symbol des Papierkorbs.
Zur Sicherheit werden Sie noch einmal gefragt, ob Sie diesen Eintrag tatsächlich löschen möchten. Sollte dies der Fall sein, so bestätigen Sie die Meldung und der Eintrag wird aus dem System gelöscht.

5. Einen Eintrag ändern

Möchten Sie einen Eintrag in Ihrer Datenbank ändern, so können Sie das im Register Orders erledigen.
Dort sehen Sie alle Einträge, die Sie bisher in Ihre Datenbank gespeichert haben.
Um Details an einem bestimmten Eintrag bzw. Verkauf zu ändern, navigieren Sie zuerst zu dem Verkauf, den Sie ändern möchten und klicken dort auf den Stift. Nun öffnet sich eine neue Seite, in der Sie alle Details zu Ihrem Verkauf sehen und komfortabel ändern können. Haben Sie Änderungen vorgenommen und möchten diese speichern, so klicken Sie auf "Aktualisieren" und der Eintrag in Ihrer Datenbank ändert sich.
